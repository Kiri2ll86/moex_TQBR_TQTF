# moex_TQBR_TQTF
Программа по заданным тикерам рублевых акций(TQBR) и фондов(TQTF), торгующихся на московской бирже отслеживает их цену на конец торговой сессии. 
Программа формирует .xlsx файлы отдельно для каждого тикера, где можно просмотреть как менялась стоимость акций за всю историю торгов в данном режиме, удобно для формирования своего инвестиционного протфеля и отслеживания кореляции между активами.____
